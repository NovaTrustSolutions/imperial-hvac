# Imperial HVAC — Engineering Consultation & Booking Page

A high-fidelity contact and booking page for "Imperial HVAC" that streamlines client inquiries while maintaining the premium industrial aesthetic.

**DESIGN SYSTEM (REQUIRED):**
- **Platform:** Web, Desktop-first
- **Theme:** Dark, high-end, industrial-chic.
- **Background:** Deep Charcoal (#1A1A1A).
- **Primary Accent:** Electric Cobalt (#0055FF) for buttons and icons.
- **Secondary Accent:** Brushed Silver (#E0E0E0) for lines, borders, and monospaced text.
- **Typography:** Space Grotesk (Headings).

**Page Structure:**
1. **Glass Header:** Consistent navigation with Logo.
2. **Consultation Brief:** Headline: "Request a Technical Consultation." Subtext: "Precision diagnostics start with a conversation."
3. **The Booking Machine:** A highly organized, technical-looking form. Use blueprint-style dividers between groups:
    - **Unit Identity:** (Name, Phone, Email)
    - **System Diagnostics:** (Type of service selected: Heating, Cooling, Ventilation, Smart Home)
    - **Terrain Mapping:** (Address and Property details)
4. **Direct Frequency:** (Alternative contact info: Direct Line, Technical Support Email, Headquarters Address).
5. **Technical Footer:** Constant from index/services.

---
💡 **Instruction:** The form should feel like an official engineering request document. Use thin lines, blueprint iconography, and high-contrast input fields.
